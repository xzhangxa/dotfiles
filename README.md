# Dotfiles

Personal shell config files for Debian/Ubuntu and macOS. See setup scripts for details.

## Setup

- `setup_debian.sh` тАФ Debian/Ubuntu
- `setup_mac.sh` тАФ macOS
- `setup_kernel_dev.sh` тАФ kernel dev extras
- `setup_oneapi.sh` тАФ Intel oneAPI
- `setup_desktop.sh` - Desktop environment

## Configs

- **neovim** (`nvim/`)
- **zsh** (`zshrc`)
- **tmux** (`tmux.conf`)
- **git + lazygit** (`gitconfig`, `lazygit.yml`, `git-proxy`)
- **gdb** (`dgdb`, `gdb_dashboard`)
- **starship** (`starship.toml`)
- **keyd** (`keyd-default.conf`, `local-overrides.quirks`)
- **opencode** (`opencode/`)
- **kitty** (`kitty.conf`)

## Tools Installed not from official distribution repository

- fzf, ripgrep, bat, eza, fd-find, uv
